# Predictive Analysis of Renewable Energy Production in Wales

## Overview

This project was completed as part of my MSc Data Science dissertation at the University of South Wales.

The project analyses renewable energy generation data in Wales and develops forecasting models using Python and ARIMA techniques. Interactive dashboards were also developed using Power BI.

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- ARIMA
- Power BI

## Project Components

- Data Cleaning and Preprocessing
- Exploratory Data Analysis (EDA)
- Correlation Analysis
- Renewable Energy Forecasting
- Power BI Dashboard
